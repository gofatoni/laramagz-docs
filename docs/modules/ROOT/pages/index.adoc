= Getting Started

== Server Requirements

* PHP >= 8.0
* MySQL 5.6+ or MariaDB 10.0+
* BCMath PHP Extension
* Ctype PHP Extension
* Fileinfo PHP extension
* JSON PHP Extension
* Mbstring PHP Extension
* OpenSSL PHP Extension
* PDO PHP Extension
* Tokenizer PHP Extension
* XML PHP Extension
* Imagick PHP Extension
* GD PHP Extension
